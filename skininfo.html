Skin features :

  1. Valid Xhtml and CSS .
  2. Fixed width. for 1024+ resolutions
  3. Tested with FF2+/3 IE 6+ Opera9+ and other browsers to comply with and cover any specific visual bugs
  4. Includes all default containers.
  5. Includes additional google code scripts to let your visitors visit your website properly making IE act like a compliant browser.Thanks to Foppe Hemminga (Afwas) for pointing to the project
  6. Includes seperate display handlers for main/single post, page mode
  7. 2 column main design, 1 column in single mode
  8. Additional comments for unordinary property/values in files to give the user idea what is going on.
